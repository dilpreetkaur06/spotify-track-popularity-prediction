About Spotify Track Popularity Prediction.

A production-style, end-to-end machine learning system that predicts a track's Spotify popularity score (0–100) from its release metadata, artist statistics and genre — built to demonstrate modular software engineering, reproducible ML pipelines and MLOps best practices.

What it predicts

Given a track's number on its album, duration, explicit flag, artist popularity & follower count, genre, album release date, album size and album type, the model estimates Spotify's track_popularity metric — a 0–100 score reflecting how much a track is currently being played relative to all other tracks on the platform.

Why it's useful

Labels, managers and independent artists can use popularity-driver insights (release timing, artist reach, genre positioning) to inform release strategy — while engineers can study this repository as a template for building modular, testable ML systems end-to-end.

Pipeline architecture

Every stage is an independently executable, OOP component under src/.

Data Ingestion→ Data Validation→ Feature Engineering→ Data Transformation→ Model Training→ Model Evaluation→ Prediction Pipeline→ Flask App

Technology stack

Python 3.10 Pandas / NumPy scikit-learn XGBoost CatBoost LightGBM MLflow DVC Flask Docker GitHub Actions
